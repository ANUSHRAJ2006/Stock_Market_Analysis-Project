# 📊 Stock Market Analysis using Python

## 🧠 Project Overview
This project analyzes global stock prices using Python, Pandas, NumPy, and Plotly to visualize trends and insights.

## ⚙️ Tools Used
- Google Colab
- Python (Pandas, NumPy, Plotly)
- Matplotlib, Seaborn

## 📁 Dataset
World_Stock_Prices_Dataset.csv

## 📈 Key Insights
- Daily and monthly trends
- Industry-wise performance
- Dividend and capital growth visualization
